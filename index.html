How to Use

Graphic Design Software

1. Download Haru600-Regular.ttf
2. For Windows: Double-click the font file to install
   For Mac: Double-click and click "Install Font"
3. Open your graphics software (Photoshop, Illustrator, etc.)
4. Create a new document
5. Select the Text tool
6. Choose "Haru600" from the font dropdown menu
7. Start creating!

CSS Implementation

@font-face {
  font-family: 'Haru600';
  src: url('Haru600-Regular.ttf') format('truetype');
}

.your-element {
  font-family: 'Haru600', sans-serif;
}
