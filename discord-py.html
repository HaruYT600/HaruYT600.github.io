How to make a discord bot

In this tutorial, you will learn how to create a simple discord bot using Python and the discord.py library.

You will need a bot account and a Discord server to follow along with this tutorial.
If you don't have a Discord server, you can create one now

Setting up your bot account

To create a bot account, you need to go to the discord developer portal and create a new application.

Bot name

Now put a name for your bot. This will be the name that appears in the server list and in the bot's profile. You can also change the bot's avatar and description later.

Bot profile

Now you can choose the bot's profile picture and description. You can also change the bot's name and avatar later.

Enable Intents

Now that you have your bot account set up, you need to enable the bot to access the necessary intents.

Intents are a way to specify what the bot is allowed to do. They are like the bot's permissions – they allow the bot to do specific things. For example, the bot needs to be able to read messages and send messages, so it needs the Read Messages and Send Messages intents.

To enable intents, go to the bot tab, scroll down to the Privileged Gateway Intents section, and enable both the Presence Intent and the Server Members Intent checkboxes. Then hit Save Changes to save your changes.

Bot token

Now that you have your bot account set up and enabled intents, you need to create a bot token. Go to the bot tab, and click the Reset Token button. This will generate a new token for your bot, which you can use to authenticate your bot to the Discord API. Remember to save this token securely, as it is your bot's password and is the only way to interact with your bot.

Bringing your bot to life

Now, you can start coding your bot! To start coding your bot, you need to install the discord.py library. You can do this by running the following command in your terminal or command prompt:

pip install discord.py

Once you have installed the library, you can start coding your bot.

Code the bot

Now, you need to create a main.py file on your python project.



In your main.py file, you need to import the discord.py library and create a new bot instance.

        
import discord
from discord.ext import commands
          
intents = discord.Intents.all()
bot = commands.Bot(command_prefix='!', intents=intents)

Now, you need to create a event listener for the on_ready event. This event is called when the bot is ready (online)

          
@bot.event
async def on_ready():
    print(f'Logged in as {bot.user.name}')

Now, you need to create a command for your bot. To create a command, you need use the @bot.command() decorator above a function. This decorator tells the bot that this function is a command.


The function should take ctx (contex) as its first parameter, representing the invocation context



For example, let's create a command that when user sends !hello the bot responds with Hello, world!

          
@bot.command()
async def hello(ctx):
    await ctx.send("Hello, world!")

Now, you need to run your bot. To run your bot, you need to use the bot.run() method. This method takes your bot token as its argument, which you can get from the Discord Developer Portal.

          
bot.run('YOUR_BOT_TOKEN')

Now, you can run your bot and test it out. To test your bot, you need to invite the bot to your server.
To add the bot to your server, you need to have the Manage Server permission.
Then you go to the Discord Developer Portal and click on the OAuth2 tab. Then, you need to select the bot scope and the Send Messages permission. Then, you can copy the link that appears and paste it in your browser to invite the bot. Once you have invited the bot, you can start chatting with it in your server.

Message Events

If you want to create event that listens to messages, you need to use the on_message event. This event is called when a message is sent in a server or a DM. You can use the message.content property to get the content of the message.

            
@bot.event
async def on_message(message):
    if message.author == bot.user:
        return  # Ignore messages from the bot itself

    if message.content.startswith('!hello'):
       await message.channel.send('Hello!')

User joins server event

If you want to create event that listens to user joins server, you need to use the on_member_join event. This event is called when a user joins a server. 

              
@bot.event
async def on_member_join(member):
    welcome_channel_id = 123456789012345678  # Replace with your welcome channel ID
    welcome_channel = bot.get_channel(welcome_channel_id)

    if welcome_channel:
        await welcome_channel.send(f'Welcome, {member.mention}!')
